As I put together this little blog and other web platforms I've tried to keep assets lightweight. I do this because I want the page to load quickly, but also because like low-tech magazine I recognize that bandwidth uses electricity. If I can put in a little bit of effort to reduce file sizes so that I can increase site performance and be more energy efficient you're damn right I'm doing it. I'm also very inters>ted in photography, which is where the banner image for this post came from. As well as computer graphics, what do web development, photography, and computer graphics have in common you may ask? Well people in all fields become familiar and argue about how images and data are encoded and decoded. They all have different requirements but for me it means I have a slight obsession with file types.
